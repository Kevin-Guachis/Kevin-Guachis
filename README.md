## 📌 Proyectos destacados

### 🎓 Sistema de Gestión de Calificaciones - Conservatorio Nacional de Música

Sistema web desarrollado para la gestión académica de calificaciones, parciales, quimestrales, notas finales e información estudiantil.

**Tecnologías:** React, Node.js, JavaScript, SQL, Bootstrap  

**Funciones principales:**
- Gestión de calificaciones por parciales y quimestres.
- Administración de información académica.
- Control de notas finales.
- Interfaz web para usuarios académicos.
- Separación entre frontend y backend.

Repositorios:
- Frontend: [Proyecto_CNM_Frontend](https://github.com/Kevin-Guachis/Proyecto_CNM_Frontend)
- Backend: [Proyecto_CNM_BackEnd](https://github.com/Kevin-Guachis/Proyecto_CNM_BackEnd)
